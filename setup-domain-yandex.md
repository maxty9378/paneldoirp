# Подключение домена к Яндекс.Облако ВМ

## Шаг 1: Настройка DNS у регистратора домена

У вашего регистратора домена (например, reg.ru, nic.ru, timeweb.ru) нужно добавить DNS записи:

### Вариант A: Поддомен (рекомендуется)
```
Тип: A
Имя: api (или supabase, или любое другое)
Значение: 51.250.94.103
TTL: 3600
```

Или для корневого домена:
```
Тип: A
Имя: @ (или пусто)
Значение: 51.250.94.103
TTL: 3600
```

### Вариант B: CNAME (если есть статический IP)
```
Тип: CNAME
Имя: api
Значение: vm-doirp.ru-central1.internal
```

## Шаг 2: Установка и настройка Nginx на ВМ

Nginx будет проксировать запросы к Supabase и вашему приложению.

## Шаг 3: Настройка SSL сертификата (Let's Encrypt)

Для HTTPS нужно получить бесплатный SSL сертификат.

## Шаг 4: Обновление конфигурации Supabase

После подключения домена нужно обновить `SITE_URL` в Supabase.
